# Installation and setup
This app can be installed through KLayout package manager
After installation the app can be accessed through toolbar, split shape icon

# Functions 
* Horizontal split
* Vertical split
* Cross split
* Center split
https://www.klayout.de/forum/discussion/894/slice-a-shape-into-two-parts#latest


### Shape Split

<p align="center">
<img align="middle" src="https://github.com/s910324/SplitShapePlugin/assets/1561043/18e17b20-a371-4f04-8a15-aadf2811dc1d" alt="installation" width="800"/>

<p align="center">

* Select one shape for split
* Click split shape icon to activate the plugin
* Use mouse right ight click to cycle between each split mode
* Left click to apply change
